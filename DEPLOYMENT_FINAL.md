# Final Deployment Steps (Real‑Data Ready)

This document ties together the free toolchain described in the roadmap and the existing codebase, providing a checklist and simple scripts so you can spin up the entire system with live Stockbit data.

---

## Prerequisites

1. Clone the repository and install dependencies:
   ```powershell
   git clone https://github.com/your/repo.git c:\IDX_Analyst
   cd c:\IDX_Analyst
   python -m venv .venv
   & .\.venv\Scripts\Activate.ps1
   pip install -r apps/ml-engine/requirements.txt
   cd apps/web && npm install && cd ../..
   go mod tidy
   ```
2. Create a Supabase project and push schema (see `DEPLOYMENT.md` section _Free Tools & Provider Setup_).
3. Create a Vercel project pointing at `apps/web` and configure environment variables.
4. Obtain a Gemini API key and save it in the environment.
5. Use the Chrome extension to capture a current Stockbit bearer token.
